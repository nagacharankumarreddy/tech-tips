# How to run this application

Once you have Node.js and npm installed, you can use the following steps to run this React app:

Navigate to the root directory of your cloned app in your terminal or command prompt.
Run the following command to install the dependencies for your app:

### `npm install`

Once the dependencies are installed, you can run the app with the following command:

### `npm start`

This should start the development server for your React app and automatically open a new browser window with your app running. If the browser window does not open automatically, you can visit http://localhost:3000/ to see your app in action.
